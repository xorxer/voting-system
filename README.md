# Installation
## Frontend
```
cd frontend
pnpm install
```
## Backend
```
cd backend
pnpm install
```
## MetaMask
Install the [MetaMask](https://metamask.io/) extension
# Running the project
## Frontend
```
pnpm run dev
```
## Backend
### Locally
```
pnpm run start
```
### Production
```
pnpm run dev
```
